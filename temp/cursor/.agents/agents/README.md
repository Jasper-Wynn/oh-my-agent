# 自定义 Agent 定义

> **作用**: Agent 行为定义和配置
> **工具**: Cursor
> **原生文档**: [Agent Overview](https://docs.cursor.com/docs/agent/overview)

## 说明

Cursor 支持 Agent 模式，包括：
- **Agent Mode**: 智能任务执行模式
- **Subagents**: 子代理用于隔离处理特定任务
- **Cloud Agents**: 云端代理能力
- **Plan Mode**: 计划模式，先制定计划再执行
