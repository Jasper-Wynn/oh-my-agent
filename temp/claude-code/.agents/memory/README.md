# Memory Bank

> **作用**: 持久化记忆 / 项目上下文
> **工具**: Claude Code
> **类型**: 状态记录 / 决策日志

## 影响范围

- **作用域**: 跨会话持久化
- **加载时机**: 会话开始时自动加载（部分工具）
- **用途**: 记录决策、进度、上下文，避免重复沟通

## 文件说明

| 文件 | 用途 |
|------|------|
| `activeContext.md` | 当前会话的活跃上下文 |
| `productContext.md` | 产品背景和业务逻辑 |
| `progress.md` | 当前进度和待办事项 |
| `decisionLog.md` | 架构决策记录 (ADR) |

## 格式说明

```markdown
# Active Context

## 当前任务
- 正在实现: xxx
- 阻塞项: 无

## 最近决策
- 决策 1: 原因...

## 待确认
- 问题 1
```

## 注意事项

- 部分工具自动维护记忆，无需手动编辑
- 手动维护的记忆应定期更新，避免过时
