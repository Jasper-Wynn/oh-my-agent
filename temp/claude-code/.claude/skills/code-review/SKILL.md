# Code Review Skill

## Checklist
- [ ] Logic correctness
- [ ] Error handling
- [ ] Test coverage
- [ ] Security considerations
- [ ] Performance implications

## Process
1. Read the changed files
2. Understand the context and intent
3. Check for edge cases
4. Verify test coverage
5. Provide actionable feedback
